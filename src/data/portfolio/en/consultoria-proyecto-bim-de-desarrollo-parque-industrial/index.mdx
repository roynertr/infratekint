**Location:** Duarte Highway, Pedro Brand

**Client:** Confidential

**Intervention Area:** 230,000 sqm

**Stage:** Preliminary Design

**Company Roles:**

- **Technology Consulting:** Advised on tech solutions for existing data collection.

- **Digital Survey Alternatives Assessment:** Evaluated technological alternatives for digital land surveying and point cloud generation.

- **BIM Data Development Options:** Reviewed options for creating BIM data to support project development.

- **Preliminary Design Guidance:** Guided the preliminary design phase with ISO 19650 standards implementation.

- **Project and Asset Requirements Gathering:** Organized project requirements and assets in alignment with ISO 19650 standards.

- **BIM Execution Plan Development:** Crafted a BIM Execution Plan tailored to client specifications and project needs.
