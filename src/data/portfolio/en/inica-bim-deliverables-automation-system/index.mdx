**Industry:** Tourism Hotel

**Service:** BIM Process Automation

**Location:** Punta Cana, Dominican Republic

## Overview

In June 2026, INFRATEK completed a BIM automation project for INICA supporting the production of deliverables for the Moon Palace Punta Cana project. The engagement focused on reducing repetitive manual work between field data collection, BIM production, and the preparation of Excel-based reports.

The solution combined Autodesk Construction Cloud, Revit and Dynamo automation, and Python-based data processing. It standardized the creation of views and sheets by system segment, extracted model information, incorporated exported images into the client’s existing Excel templates, and established a structured field-to-office workflow. The system was validated with real project data and prepared for continued use across active contracts.

![BIM Deliverables Automation Workflow — from field capture to structured reporting](../inica-bim-deliverables-automation-system/workflow.png)

## Deliverables

Final deliverables included:

- Modular Revit and Dynamo automation workflows for views, sheets, tagging, and data export
- Automated Excel reports integrating model data, sketches, key plans, and exported images
- ACC folder structure, field capture workflow, installation documentation, and user guidance

## Services Provided

- BIM Process Automation
- Revit, Dynamo, and Python Development
- BIM Data Integration and Automated Reporting

This project reflects INFRATEK’s capability to transform repetitive BIM production processes into scalable, standardized, and operational digital workflows.
