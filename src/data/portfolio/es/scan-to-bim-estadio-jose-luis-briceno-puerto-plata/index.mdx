## Resumen

En febrero de 2023, INFRATEK desarrolló un proyecto Scan-to-BIM para el Estadio de Baseball José Luis Briceño, ubicado en Puerto Plata, República Dominicana. El proyecto fue realizado a partir de una nube de puntos suministrada por el cliente para apoyar procesos de documentación digital, modelado de condiciones existentes y visualización BIM del activo.

El alcance incluyó la interpretación y procesamiento de información proveniente de escaneo láser para generar un modelo BIM coordinado representando las condiciones arquitectónicas y estructurales del estadio. El modelo fue desarrollado para facilitar flujos de coordinación digital e interoperabilidad bajo estándares Open BIM.

## Entregables

Los entregables finales incluyeron:

- Modelo nativo en Autodesk Revit
- Exportación IFC4 para interoperabilidad Open BIM
- Documentación en formato DWG
- Archivos complementarios para coordinación y visualización

## Servicios Realizados

- Modelado Scan-to-BIM
- Interpretación de Nube de Puntos
- Modelado de Condiciones Existentes
- Entrega Open BIM / IFC4
- Modelado en Revit
- Documentación Digital

Este proyecto demuestra la capacidad de INFRATEK para transformar datos complejos de reality capture en activos BIM confiables para entornos de infraestructura y edificaciones.
